    Object.assign()

    js中存储对象存的是地址，所以浅拷贝是都指向同一块内存区域，而深拷贝则是开辟来一块区域，浅拷贝在改变其中一个值的时候会导致其他的对象也一起改变，
    而深拷贝则不会，Object.assign()只是一级属性复制，比浅拷贝多拷贝了一层。
    const obj1 = JSON.parse(JSON.stringify(obj))也可以实现深拷贝。
