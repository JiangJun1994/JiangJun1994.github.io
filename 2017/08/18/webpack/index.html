    • webpack 配置

    webpack是一个前端资源加载／打包工具，它将根据模块的依赖关系进行静态分析，然后将模块按照指定的规则生成对应的静态资源。
    webpack 可以将多种静态资源（js,css,less）转换成一个静态文件，减少页面的请求。

    安装

    安装之前确保本地环境支持node.js

    loader

    webpack本身只能处理js模块，如果要处理其他类型的文件，就需要loader去进行转换。
    如果我们需要在应用中添加css文件，就需要css-loader和style-loader,css-loader 会遍历 CSS 文件，
    然后找到 url() 表达式然后处理他们，style-loader 会把原来的 CSS 代码插入页面中的一个 style 标签中。

    插件

    插件在 webpack 的配置信息 plugins 选项中指定，用于完成一些 loader 不能完成的工。
    loader 是在打包过程中用来处理源文件的，一次处理一个，插件并不直接操作单个文件，它直接对整个构造过程起作用。
