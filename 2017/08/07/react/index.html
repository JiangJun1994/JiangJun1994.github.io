    父子组件之间传值

    父组件向子组件传值

    父组件向子组件传值主要是通过props;

    子组件通过委托向父组件传值

    react中state改变啦，组件就会更新，父组件写好state和处理该state的函数，
    父组件的处理函数通过属性的方式赋值给子组件，子组件通过触发事件，委托调用父组件的处理函数，同时引起state的变化，从而实现把值传给父组件;
